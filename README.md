# cppExploitV2 — MD5 爆破工具（1digi 分支）

> **分支说明**：本分支临时修改为**专门测试手机号**，穷举所有 `1` 开头的 11 位电话号码。

## 哈希格式

```
hash = MD5( MD5(password) + MD5("hugo") ).substr(8, 16)
```

即：密码的 MD5 与固定 salt `"hugo"` 的 MD5 拼接后，再做一次 MD5，取中间 16 位。

## 算法

- 基于 **n 进制计数器** 穷举所有字符组合
- 字符集：数字 `0-9`
- **手机号模式**：实际测试的密码为 `"1" + str`（在 MD5 计算前拼接），`str` 为程序穷举的 10 位数字
- 多线程：按**首字符**拆分任务，每个线程枚举以 `asciis[id]` 开头的所有组合
- **提前终止**：某个线程找到密码后，立即通知其他线程退出，避免浪费 CPU
- **进度条**：监控线程每 15 秒刷新一次进度和预估剩余时间，开销趋近于零

## 文件

| 文件 | 说明 |
|------|------|
| `main.cpp` | 主程序：爆破入口 + 多线程调度 + 穷举核心（已修改为测试手机号） |
| `md5.cpp` / `md5.h` | MD5 类实现（JieweiWei） |
| `md5_origin.cpp` | 函数式 MD5 实现（备选） |
| `gen.cpp` | 快速验证工具 |

## 编译

```bash
g++ main.cpp -o exploit.exe -std=c++11 -O2 -lpthread
```

## 使用

### 命令行参数

```
-h, --hash <hash>      目标哈希（缺省则交互式询问）
-l, --length <length>  密码长度（缺省则交互式询问）
-o, --output <file>    结果写入文件（缺省则打印到控制台）
```

### 示例

```bash
# 爆破手机号：length 设为 10（10 位变化数字，前面自动补 "1"，共 11 位手机号）
./exploit.exe -h a1b2c3d4e5f6a1b2c3d4 -l 10 -o result.txt

# 交互式
./exploit.exe
hash: a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4
length: 10
```

> ⚠️ 注意：`-l` 必须传入 **10**，程序实际测试的是 `"1" + 10位数字` = 完整的 11 位手机号。

### 进度条

运行时进度条示例（每 15 秒刷新）：

```
Progress: [##############      ]  70%  12m34s  70000000/100000000
```

格式：`进度条 | 百分比 | 预估剩余时间 | 已完成/总计`

## 与主分支的区别

| 特性 | main | 1digi（本分支） |
|------|------|----------------|
| 测试密码 | `str`（原始穷举组合） | `"1" + str`（手机号） |
| 典型长度 | ≤ 6 位 | 10 位（生成 11 位手机号） |
| 用途 | 通用短密码爆破 | 专测手机号哈希 |
